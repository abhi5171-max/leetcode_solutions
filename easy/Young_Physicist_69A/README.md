2. Young Physicist
Problem Summary

A body is acted upon by several force vectors in three-dimensional space. Determine whether the body is in equilibrium by checking if the sum of all force vectors equals the zero vector.

Approach
Maintain three variables to store the sum of the x, y, and z components.
Add each vector's components to their respective sums.
If all three sums are zero, the body is in equilibrium.
Algorithm
Initialize x_sum, y_sum, and z_sum to 0.
Read each force vector.
Add its components to the corresponding sums.
If all sums are zero, print "YES".
Otherwise, print "NO".
Time Complexity
O(n)
Space Complexity
O(1)
Concepts Used
Vectors
Accumulation
Simulation
