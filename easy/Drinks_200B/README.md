## Problem 2: Drinks

### Problem
Given the percentage of orange juice in `n` drinks, find the percentage of orange juice in a cocktail made by mixing **equal amounts** of all the drinks.

### Approach
Since equal quantities of every drink are mixed, the final orange juice percentage is simply the **average** of all the given percentages.

### Algorithm
1. Read `n`.
2. Read the percentages of all drinks.
3. Calculate their sum.
4. Divide the sum by `n`.
5. Print the result.

### Complexity
- **Time:** `O(n)`
- **Space:** `O(n)`

### Key Concepts
- Average Calculation
- Arrays/Lists
- Basic Mathematics
